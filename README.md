# Java Attendance Manager

A CLI-based Attendance Management System built using **Core Java, OOP, File Handling, and Multithreading**.

## 🚀 Features

* Manage attendance for multiple subjects
* Calculate attendance percentage
* File-based storage system
* Multithreaded report generation
* JavaDoc documentation support

## 🧠 Concepts Used

* Core Java
* Object-Oriented Programming (OOP)
* File Handling
* Multithreading
* JavaDoc

## 📁 Project Structure

AttendanceManager
│
├── Student.java
├── Subject.java
├── AttendanceService.java
├── FileManager.java
├── ReportGenerator.java
└── Main.java

## ▶️ How to Run

Compile the program:

javac *.java

Run the program:

java Main

## 📚 Documentation

JavaDoc documentation can be generated using:

javadoc -d docs *.java
